Реализация чата выполнена под систему Linux.
Все действия, указанные ниже нужно выполнять в Linux.
Для работы чата требуется установить библиотеку libmysqlclient-dev , приложение mysql
Ниже приведены команды и ключи, которые необходимо ввести для успешной сборки и компиляции.

 sudo apt-get install libmysqlclient-dev 
 sudo apt-get install mysql 
 sudo apt install mysql-server mysql-client

Если у вас уже установлен mysql, вам нужно изменить значения в файле mysql_interface.h
начиная с 12 строки на ваши данные. 

По умолчанию значения переменных равны 

const std::string DEFAULT_DATABASE_NAME = "my_database";
const std::string DEFAULT_PASSWORD_TO_DATABASE = "412313";
const std::string DEFAULT_USERNAME_FOR_DATABASE = "root";
const std::string DEFAULT_HOSTNAME = "localhost";

Порты, используемые чатом по умолчанию равны 8080. Указаны в файлах client_to_server_req.h и server_interface.h .


Для начала вам нужно скомпилировать два файла. Одна программа - клиент, вторая сервер. Приложения 
собираются и комплируются командами (сборку рекомендуется выполнять из-под администратора)

сервер: 

g++ -std=c++17  -o server.app test_of_server_17_05.cpp  -lm -pthread -lmysqlclient


клиент:

 g++ -std=c++17  -o client.app test_of_client_17_05.cpp


 После сборки вам нужно запустить сервер, только потом клиент, причем каждое приложение 
 запускается в отдельном окне. Если запуск из обычного пользователя не сработал или выдает
 ошибки, вам нужно запустить приложения из суперпользователя. 

 После запуска сервера, в окне будут отображаться некоторые статистические данные.
 В окне клиента будет меню для взаимодействия. Первым делом вам нужно зарегистрироваться
 для этого напишите /reg и нажмите enter. 

 После этого вам будет предложено ввести имя пользователя, пароль и т.д.

 После ввода всех данных вам будет предложено зарегистрироваться, залогиниться или выйти.
 Для отправки и получения сообщений вам нужно зарегистрировать еще одного пользователя. 
 Выполните то же самое, отправьте /reg и введите данные. 

 После регистрации второго пользователя вы можете залогиниться. 
 Для этого введите /login и выполняйте указания. 

Если сервер не отвечает, перезапустите приложение и сразу выберите /login .

 После успешного входа появится меню, в котором будут предложены команды 
 /users   ... команда показывает всех зарегистрированных пользователей чата
 /message  ... команда для отправка сообщения конкретному пользователю
 /read     ... команда для чтения сообщения от конкретного пользователя
 /exit     ... команда для выхода.
